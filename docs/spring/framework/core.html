Core Technologies

참조 문서의 이 부분은 Spring Framework에 절대적으로 필수적인 모든 기술을 다룹니다

가장 중요한 것은 Spring Framework의 Inversion of Control (IoC) container입니다. Spring Framework의 IoC container에 대한 철저한 처리는 Spring의 AOP (Aspect-Oriented Programming) 기술에 대한 포괄적인 coverage에 의해 밀접하게 뒤따릅니다. Spring Framework에는 개념적으로 이해하기 쉽고 Java enterprise programming에서 AOP 요구 사항의 80% sweet spot을 성공적으로 해결하는 자체 AOP framework가 있습니다.

Spring의 AspectJ와의 통합(현재 기능 측면에서 가장 풍부하고 Java enterprise space에서 가장 성숙한 AOP 구현)에 대한 coverage도 제공됩니다.
